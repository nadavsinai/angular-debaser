debaser(name, opts) → {Debaser}

Provides a Debaser object with which you can stub dependencies easily.

Parameters:
Name Type Argument Default Description
name String | Object <optional>
 __default__

Optional name of Debaser. Only useful if using multiple instances. If omitted, this is considered the opts parameter.
opts Object <optional>

Options to modify angular-debaser's behavior.

Properties
Name Type Argument Default Description
debugEnabled Boolean <optional>
 false

Whether or not to show debug output.
autoScope Boolean <optional>
 true

Whether or not to provide Scope objects automatically to the $controller service.
skipConfigs Boolean <optional>
 true

Whether or not to skip config() blocks when loading existing modules.
